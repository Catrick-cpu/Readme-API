🎨 GitHub README Preview API

✅ API is live and ready!

Available Endpoints

List All Renderers GET /api/render-v2?list=true
Get Renderer Metadata GET /api/render-v2?catalog=true
Render Content GET /api/render-v2?type=markdown&content=%23%20Hello

19 Renderers Available

  • Documentation: markdown, changelog, readme-section
  • Code: code, json, yaml, terminal, diff
  • Data: csv, table, bar-chart, progress, stats
  • Diagrams: mermaid, timeline
  • Components: badge, button, card, divider
